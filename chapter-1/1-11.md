# 练习 1.11

函数 `f` 由如下的规则定义：如果 `n < 3`，那么 `f(n) = n`；如果 `n >= 3`，那么 `f(n) = f(n-1) + 2f(n-2) + 3f(n-3)`。请写下一个采用递归计算过程计算 `f` 的过程。再写一个采用迭代计算过程计算 `f` 的过程。

---

## 递归过程

```bash
(define (f n)
  (if (< n 3) n
      (+ (f (- n 1)) (* 2 (f (- n 2))) (* 3 (f (- n 3))))))
```

## 迭代过程

```bash
(define (f n)
  (f-iter 0 1 2 n))

(define (f-iter a b c n)
  (cond ((= n 0) 0)
        ((= n 1) 1)
        ((= n 2) c)
        (else (f-iter b c (+ c (* 2 b) (* 3 a)) (- n 1)))))
```
