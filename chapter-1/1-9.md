# 练习 1.9

下面几个过程各定义了一种加起两个正整数的方法，它们都给予过程 `inc`（它将参数增加 1）和 `dec`（它将参数减少 1）。

```bash
(define (+ a b)
  (if (= a 0)
      b
      (inc (+ (dec a) b))))

(define (+ a b)
  (if (= a 0)
      b
      (+ (dec a) (inc b))))
```

请用代换模型展示这两个过程在求值 `(+ 4 5)` 时所产生的计算过程。这些计算过程是递归的或者迭代的吗？

---

## 代换过程 1

```bash
(+ 4 5)
(inc (+ 3 5))
(inc (inc (+ 2 5)))
(inc (inc (inc (+ 1 5))))
(inc (inc (inc (inc (+ 0 5)))))
(inc (inc (inc (inc 5))))
(inc (inc (inc 6)))
(inc (inc 7))
(inc 8)
9
```

代换过程 1 是递归计算过程。

## 代换过程 2

```bash
(+ 4 5)
(+ 3 6)
(+ 2 7)
(+ 1 8)
(+ 0 9)
9
```

代换过程 2 是迭代计算过程。
